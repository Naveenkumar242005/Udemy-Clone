# Udemy Clone Website

A simple Udemy-inspired course platform frontend built using HTML5 and CSS3. This project was created to practice web development fundamentals, including page structure, layouts, styling, and responsive design concepts.

## Features

- Udemy-style homepage layout
- Course cards with images, ratings, and pricing
- Navigation bar
- Search section
- Responsive design basics
- Clean and modern user interface

## Technologies Used

- HTML5
- CSS3

## Purpose

This project was developed as a learning exercise to improve frontend development skills and gain hands-on experience with HTML and CSS.
